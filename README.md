A location-based multiplayer game where players send snails to each other's home bases in Singapore.

## What is it?

The Snail is a real-time geolocation game where you:
- Set up a home base
- Deploy snails to travel to your friends' locations
- Intercept incoming snails before they reach your base
- Earn salt currency and manage your snails

Snails takes days to travel to their end location, squirming along real walking paths.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Maps**: Mapbox GL JS
- **Backend**: Supabase (PostgreSQL + PostGIS)
